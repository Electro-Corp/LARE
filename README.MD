# LARE
## Lightweight Advanced Rendering Engine
LARE is a lightweight rendering engine utilizing OpenGL that is built to be flexible, powerful, fast to prototype, and easily extendable. <br>

## Build
```
mkdir build
cd build
cmake ../
make
```

## Third Party Stuff

- Assimp 
- Lua 5.1
